# Currency Converter / 通貨コンバーター

## Description / 説明
An application to convert currency between USD, BDT, and JPY.
USD・BDT・JPY間で通貨を変換するアプリケーションです。

## Features / 機能
- Supports 3 currencies: USD, BDT, JPY /  3つの通貨に対応：USD・BDT・JPY
- Uses fixed exchange rates / 固定為替レートを使用
- Simple terminal interface / シンプルなターミナル画面

## How to run / 実行方法
python3 converter.py
select from (currency) -> to (currency) 
 $-> From -> To
 $-> input your amount (From-box) -> output currency(To-box) # get expected currency amount
