# Design Plan / デザイン計画

## Terminal UI Flow / ターミナル画面の流れ

=== Currency Converter / 通貨コンバーター ===

Select your currency / 変換元の通貨を選んでください:
1. USD (Dollar / ドル)
2. BDT (Taka / タカ)
3. JPY (Yen / 円)

Select convert currency / 変換先の通貨を選んでください:
1. USD (Dollar / ドル)
2. BDT (Taka / タカ)
3. JPY (Yen / 円)

Input the amount / 金額を入力してください: (not negative / マイナス不可)

Result / 結果:
[From currency]: [amount]
[To currency / 変換先]: [converted amount]

Do you want to change the amount? / 金額を変更しますか？ (y/n)
Do you want to change currency? / 通貨を変更しますか？ (y/n)
Are you sure to finish? / 終了しますか？ (y/n)

Thank you for using Currency Converter!
ご利用ありがとうございました！
